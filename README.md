Page Studio
=============

A lightweight  PHP CMS built on an Open Source technology.

#Website
https://www.pagestudio.co/

#Requirements
A web host or server with:
* PHP >= 5.4.0
* MySQL >= 4.x

A modern and updated web browser

#Documentation
http://docs.pagestudio.co/2.0/
